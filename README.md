# 少儿编程教学作品集
你好，我是张武兰,计算机本科毕业,持有小学数学教师资格证,有2年一线教学经验。
## 这个仓库有什么?
这里存放了我为**小学四年级以上**学员设计的编程教学案例,重在体现**把抽象逻辑变生活化比喻**的教学能力。
### 1.贪吃蛇游戏(snake_game)
-**技术栈**:Python+Pygame
-**教学知识点**:坐标、数组、碰撞检测
-**课堂导入金句**:"同学们,今天我们不做游戏,我们来当游戏的设计师,指挥一条小蛇去觅食!"

### 2.猜数字游戏(guess_number)
-**技术栈**：原生Python(控制台)
-**教学知识点**:变量、循环、条件判断
-**设计亮点**:引入"二分法"数学策略,让孩子在玩中理解"最坏情况"和"最优解"。
